An Interesting Command in MacOS

Nov 1st, 2011

Today i just knew a funny command in mac os, it is command say.

This command can convert the input text to audible speech, for example :

say hello world

you can hear a male voice to read it .Also you can let it read a file using “-f file_name”, also you can record the voice to the file using “-o output.aiff ” .

The most interesting is that using “-v voice” to use different voice, for example:

say -v good hello world #Â high pitch

say -v bad hello world #low pitch
